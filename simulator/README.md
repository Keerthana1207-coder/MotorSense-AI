# MotorSense-AI

## 🚀 AI Predictive Maintenance for Industrial Motors

MotorSense-AI is an IoT + AI based predictive maintenance system that monitors industrial motor health using simulated sensor data and predicts failures before they occur.

## Tech Stack

- React
- Python
- MQTT
- Firebase
- Node-RED
- Scikit-learn
- ESP32 (Future Hardware)
